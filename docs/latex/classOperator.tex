\doxysubsection{Detailed Description}
Class representing an operator in a quantum system. 

This class provides methods for initializing, manipulating, and diagonalizing operators represented as sparse matrices. \begin{DoxyWarning}{Warning}
This class is not thread-\/safe. 
\end{DoxyWarning}
